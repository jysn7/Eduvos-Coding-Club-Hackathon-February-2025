What we offer

HTML, CSS & JS

Learn to build and style websites from scratch using HTML, CSS, and JavaScript. We’ll cover the fundamentals of web development, including structuring web pages with HTML, designing with CSS (layouts, animations, and responsiveness), and adding interactivity with JavaScript

JAVA

Learn the fundamentals of Java, from syntax and object-oriented programming to building real-world applications. We’ll cover variables, loops, methods, classes, and more, giving you the skills to write efficient, structured code. Perfect for beginners and those looking to level up!

C#

The Eduvos Coding Club is the place to dive into C# programming, whether you're just starting out or sharpening your skills. You'll learn core concepts like variables, loops, functions, and object-oriented programming, plus how to build real applications. Join us to code, create, and collaborate!

Project Portfolio Building

We are a hands-on club focused on creating real-world projects to showcase your skills. Whether you're into web development, or software engineering, you'll work on personalized projects, refine your coding abilities, and build a standout portfolio. Perfect for anyone looking to gain experience and impress future employers!

Collaborative Projects

We're centered around collaborative projects, where you’ll work with others to build real-world applications, websites, and software. Learn how to write clean code, use version control, and develop teamwork skills while creating something awesome together. Perfect for those who love to code, collaborate, and innovate!

Networking Events

The Coding Club is a networking hub for developers to meet, collaborate, and grow. Join us for events where you can connect with industry professionals, share ideas, and gain insights into the tech world. Whether you're a beginner or experienced coder, it's the perfect place to expand your network and opportunities!
